# VectorShift Frontend Technical Assessment

## Overview

This project is my submission for the VectorShift Frontend Technical Assessment.

I implemented a visual pipeline builder that allows users to:

- Drag and drop nodes
- Connect nodes visually
- Configure each node independently
- Detect dynamic variables automatically
- Validate pipeline structure
- Monitor pipeline health
- Execute pipeline validation before running

---

## Features

- React Flow based pipeline editor
- Searchable node library
- Categorized sidebar
- Reusable node architecture
- Dynamic variable detection
- Real-time pipeline validation
- DAG validation
- Pipeline health panel
- Node configuration panel
- Modern responsive UI

---

## Tech Stack

- React
- React Flow
- JavaScript
- CSS
- Zustand
